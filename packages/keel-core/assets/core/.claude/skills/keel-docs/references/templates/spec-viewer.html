No se pudo cargar el visor

El renderizado usa una librería servida desde un CDN y esta página no ha podido descargarla (normalmente, por falta de conexión). El contrato en sí está completo y embebido aquí abajo.

  • Descarga el archivo con el botón Descargar spec y ábrelo con tu herramienta habitual.
  • O pulsa Copiar spec y pégalo en el editor online.
  • O vuelve a abrir esta página con conexión a internet.
